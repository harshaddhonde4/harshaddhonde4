### 🦙 LlamaLocal RAG — Pipeline

```
  [User Input URL]
        │
        ▼
  [Webpage Fetcher] ──── LangChain Document Loader
        │
        ▼
  [Chunk Splitter] ───── 500 chars | 50 overlap
        │
        ▼
  [OllamaEmbeddings] ─── Local vector generation
        │
        ▼
  [ChromaDB] ────────── Per-URL isolated collection
        │
        ▼
  [Similarity Search] ── Retrieve top-k chunks
        │
        ▼
  [Llama 3.2 via Ollama] On-device LLM inference
        │
        ▼
  [Streamlit UI] ─────── Persistent chat history
```
